How to install NodeJS on your pi.

### Redis

Redis is needed as storage for our applications.
How to install and run Redis on your pi.

    ssh into the pi.
    wget http://download.redis.io/releases/redis-3.0.3.tar.gz
    tar -xvf redis-3.0.3.tar.gz
    cd redis-3.0.3
    make
